  • La gestion l'expérience utilisateur en temps réel ;
  • La détection automatique de la preuve de vie ;
  • La gestion de l’anti-fraude (Anti-spoofing) ;
  • La gestion de matching streaming avec le serveur Sky Deep Face Match ;

    The MRZKeyModel object contains the MRZ key information required for the NFC reader. is composed of :

    • documentNumber : the person's document number of passport or id card.
    • birthDate : the person's birth date with the format yyMMdd.
