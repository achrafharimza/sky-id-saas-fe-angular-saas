L’intégration de la SDK Mobile avec l’application mobile du client se fait via l’appel seulement à deux Activity sky_document_analyzer. Ces fonctionnalités gèrent :

Sky Document Analysis (Smart OCR) : Easy integration, une seule Activity sky_document_analyzer à appeler pour :

  • La gestion de la communication avec le Core de Sky Identification (keys, cryptage, …) ;
  • L’analyse de l’environnement client(luminosité, reflet de lumière, ...) ;
  • La gestion l'expérience utilisateur en temps réel ;
  • L’analyse de plusieurs frames par secondes (Mode streaming) pour sélectionner les meilleures images;
  • La détection automatique du ng-star-inser-star-inserocument ;
  • La vérification de la pièce d'identité ;
  • La gestion de l’OCR Streaming ;
  • Contrôler recto et verso ;
  • Contrôler l'intégrité de la pièce d'identité ;
  • Le Pré processing des images sélectionnées ;
  • Le Post processing des différents résultats obtenus ;
  • La validation des seuils et le choix du meilleur résultat d'OCÉRisation par champ et par algorithme, …
  • Le regroupement et formatage du résultat obtenu sous forme du Json ;
  • La gestion des codes retours ;
